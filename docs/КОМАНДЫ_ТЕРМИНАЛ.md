# SenimdiQAdam — Команды для терминала
## БД: локальный PostgreSQL (порт 5432, юзер alua) ✅

---

## ШАГ 1 — Остановить всё что сейчас запущено

```bash
# Если core-svc запущен — нажми Ctrl+C в том терминале
# Остановить старые Docker контейнеры с БД (они больше не нужны)
cd ~/Desktop/senim-qadam/infra
docker compose down
# Запустить только Redis
docker compose up -d
docker ps   # должен быть только senimdi_redis
```

---

## ШАГ 2 — Обновить схему Prisma для core-svc

```bash
cd ~/Desktop/senim-qadam
# Копируем новую схему (файл создан автоматически)
cp scripts/schema_core_NEW.prisma services/core-svc/prisma/schema.prisma
```

---

## ШАГ 3 — Установить пакеты и запустить миграцию core-svc

```bash
cd ~/Desktop/senim-qadam/services/core-svc

# Установить новые пакеты (JWT, Passport, bcrypt, Google OAuth)
npm install @nestjs/jwt @nestjs/passport passport passport-jwt passport-google-oauth20 bcryptjs
npm install -D @types/passport-jwt @types/passport-google-oauth20 @types/bcryptjs

# Запустить миграцию (создаст все таблицы в core_db)
npx prisma migrate dev --name add_users_auth_guides

# Сгенерировать Prisma Client
npx prisma generate
```

---

## ШАГ 4 — Запустить core-svc

```bash
cd ~/Desktop/senim-qadam/services/core-svc
npm run start:dev
# Жди: "Nest application successfully started on port 3001"
```

Проверка (в новом терминале):
```bash
curl http://localhost:3001/health
```

## ШАГ 5 — Установить пакеты и запустить taxi-svc

```bash
cd ~/Desktop/senim-qadam/services/taxi-svc

npm install
# После установки — миграция
npx prisma migrate dev --name init_taxi
npx prisma generate

# Запуск
npm run start:dev
# Жди: "Taxi-svc running on http://localhost:3002"
```

## ШАГ 7 — Настроить ai-svc (FastAPI)

```bash
cd ~/Desktop/senim-qadam/services/ai-svc

# Создать виртуальное окружение Python
python3 -m venv venv
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Запуск
uvicorn main:app --reload --port 8000
# Swagger: http://localhost:8000/docs
```

## ТЕСТ авторизации (Postman или curl)

```bash

  -H "Content-Type: application/json" \
  -d '{"email":"test@test.com","password":"Pass1234!","firstName":"Алуа","lastName":"Тест"}'

# Ответ: { "accessToken": "...", "refreshToken": "..." }

# Проверить себя
curl http://localhost:3001/auth/me \
  -H "Authorization: Bearer ВАШ_ACCESS_TOKEN"
```

## Структура портов

```
:3000  gateway-api  (точка входа для фронтенда)
:3001  core-svc     (авторизация, организации, пользователи)
:3002  taxi-svc     (бронирование такси)
:5432  PostgreSQL   (локальный: core_db, taxi_db, ai_db)
:6379  Redis        (Docker: сессии, кэш)
:8000  ai-svc       (RAG чат-бот, STT/TTS)
```
